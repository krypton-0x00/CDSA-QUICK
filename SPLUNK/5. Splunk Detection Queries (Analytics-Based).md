

## Detect Processes with Unusual Network Activity

```spl
index="main" sourcetype="WinEventLog:Sysmon" EventCode=3
| bin _time span=1h
| stats count as NetworkConnections by _time, Image
| streamstats time_window=24h avg(NetworkConnections) as avg stdev(NetworkConnections) as stdev by Image
| eval isOutlier=if(NetworkConnections > (avg + (0.5*stdev)), 1, 0)
| search isOutlier=1
```

Uses **statistical anomaly detection** to identify processes making an unusually high number of network connections compared to their normal behavior over the previous 24 hours. Useful for detecting **Command & Control (C2)** communication, malware beaconing, or data exfiltration.

## Detect Abnormally Long CMD Commands

```spl
index="main" sourcetype="WinEventLog:Sysmon" Image=*cmd.exe
| eval len=len(CommandLine)
| table User, len, CommandLine
| sort - len
```

Calculates the length of every `cmd.exe` command and lists the longest ones first. Attackers often use very long commands for obfuscation, encoded payloads, or chained execution.

---

## Detect Long CMD Commands (Reduced Noise)

```spl
index="main" sourcetype="WinEventLog:Sysmon" Image=*cmd.exe ParentImage!="*msiexec.exe" ParentImage!="*explorer.exe"
| eval len=len(CommandLine)
| table User, len, CommandLine
| sort - len
```

Filters out common benign parent processes like **Explorer** and **MSIExec** to make suspicious long command lines easier to identify.

## Detect Abnormal cmd.exe Activity

```spl
index="main" EventCode=1 (CommandLine="*cmd.exe*")
| bucket _time span=1h
| stats count as cmdCount by _time User CommandLine
| eventstats avg(cmdCount) as avg stdev(cmdCount) as stdev
| eval isOutlier=if(cmdCount > avg+1.5*stdev, 1, 0)
| search isOutlier=1
```

Uses statistical analysis to identify command lines that execute **cmd.exe** significantly more often than normal, helping detect scripts, malware, or automated attacker activity.

## Detect Processes Loading Many DLLs

```spl
index="main" EventCode=7
| bucket _time span=1h
| stats dc(ImageLoaded) as unique_dlls_loaded by _time, Image
| where unique_dlls_loaded > 3
| stats count by Image, unique_dlls_loaded
```

Detects processes that load an unusually large number of unique DLLs within an hour. Malware frequently loads many DLLs during execution.

---

## Detect Suspicious DLL Loading (Filtered)

```spl
index="main" EventCode=7 NOT (Image="C:\\Windows\\System32*") NOT (Image="C:\\Program Files (x86)*") NOT (Image="C:\\Program Files*") NOT (Image="C:\\ProgramData*") NOT (Image="C:\\Users\\waldo\\AppData*")
| bucket _time span=1h
| stats dc(ImageLoaded) as unique_dlls_loaded by _time, Image
| where unique_dlls_loaded > 3
| stats count by Image, unique_dlls_loaded
| sort - unique_dlls_loaded
```

Excludes common trusted application paths to reduce false positives and focus on potentially malicious executables loading many DLLs.

## Detect Repeated Process Executions

```spl
index="main" sourcetype="WinEventLog:Sysmon" EventCode=1
| transaction ComputerName, Image
| where mvcount(ProcessGuid) > 1
| stats count by Image, ParentImage
```

Groups process creation events and identifies programs that have been executed multiple times on the same host. Useful for spotting repeated attacker activity, persistence mechanisms, or malware loops.

---

## Investigate Repeated rundll32.exe Executions

```spl
index="main" sourcetype="WinEventLog:Sysmon" EventCode=1
| transaction ComputerName, Image
| where mvcount(ProcessGuid) > 1
| search Image="C:\\Windows\\System32\\rundll32.exe" ParentImage="C:\\Windows\\System32\\svchost.exe"
| table CommandLine, ParentCommandLine
```

Performs a deeper investigation into repeated **rundll32.exe** executions launched by **svchost.exe**. Since `rundll32.exe` is commonly abused by attackers to execute malicious DLLs (MITRE ATT&CK **T1218.011 - Signed Binary Proxy Execution**), reviewing the command line helps determine whether the activity is legitimate or malicious.
